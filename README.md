# Madalina Mesrefoglu Personal Portfolio
 
This is a responsive, single-page personal portfolio website with smooth scrolling.

## Tech Stack

- Javascript
- HTML5
- CSS3 
- AOS library
- VSCode
- GSAP (Flip plugin)
